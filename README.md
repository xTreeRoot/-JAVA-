### 可换肤MP3播放器的设计与实现

#### 目的：分析当前流行的换肤编程技术，开发一个可以管理本地音乐库的MP3播放器，要求播放器可以更换2～3种皮肤。



1. 编程实现一个可以解码MP3的音乐播放器，可以使用解码控件；
2. 播放器需要支持音乐库管理，包括本地MP3文件搜索、添加到音乐列表、从音乐列表中删除、建立音乐专辑；
3. 支持2～3种皮肤的更换；
